Warframe WorldState Dashboard
=============================

Files:
- index.html — single-file dashboard. Choose platform in the top-right and click "Обновить".
- Works with official WorldState endpoints for PC/PS4/Xbox/Switch/Mobile.
- Data updates every 30 seconds. Uses cache-busting to avoid stale content.

Notes:
- If you see HTML instead of JSON, try opening the endpoint URL directly in browser to check.
- This file is built to run from GitHub Pages (https) or locally. If served over file:// some fetch features may be limited.
